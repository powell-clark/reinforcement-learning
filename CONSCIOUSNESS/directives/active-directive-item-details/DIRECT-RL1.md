# DIRECT-RL1: Ship complete RL curriculum

## Vision

Deliver a complete, university-grade reinforcement learning curriculum — from Markov decision processes through deep RL (DQN, PPO, SAC) — with the same rigour and format as the supervised and unsupervised learning series. Every algorithm is derived from first principles, implemented from scratch in NumPy, and reproduced with a production framework (Stable-Baselines3), across paired theory and practical notebooks that run in Google Colab.

## Scope

- 16 lessons (0–15), each a paired theory (`a`) + practical (`b`) notebook: 32 notebooks.
- 4 professional-practice notebooks (X1–X4): debugging, evaluation, deployment, research frontiers.
- Cross-cutting capabilities maintained across the whole curriculum: Colab-runnable notebooks (FEAT-RL1), from-scratch NumPy + Stable-Baselines3 dual implementation (FEAT-RL2), Gymnasium environment integration (FEAT-RL3).

## Success Metrics

- Comprehensiveness: covers classical RL (Sutton & Barto) plus modern deep RL.
- Accessibility: MDPs and Bellman equations derived from first principles.
- Practicality: every algorithm runnable on Colab, scales to real problems.
- Modern: includes production algorithms (PPO, SAC).
- Hands-on: interactive Gymnasium environments, visualizations, immediate feedback.
