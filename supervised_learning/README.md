# Supervised Machine Learning Examples

A comprehensive, course-style collection of supervised learning examples with hands-on implementations.

## 📚 Contents

### 01. Regression
- **Linear Regression** - Foundation of supervised learning, gradient descent, normal equation
- **Polynomial Regression** - Modeling non-linear relationships, overfitting detection
- **Regularization** - Ridge (L2), Lasso (L1), Elastic Net for preventing overfitting

### 02. Classification
- **Logistic Regression** - Binary and multiclass classification, decision boundaries, ROC/AUC

### 03. Neural Networks (Coming Soon)
- Introduction to neural networks and deep learning

### 04. Model Evaluation (Coming Soon)
- Cross-validation, metrics, hyperparameter tuning

## 🎯 Learning Approach

Each notebook follows a consistent structure:
1. **Mathematical Foundation** - Equations and theory
2. **From-Scratch Implementation** - Build algorithms using NumPy
3. **Scikit-learn Implementation** - Production-ready code
4. **Visualization** - Understand model behavior
5. **Real-World Examples** - Apply to actual datasets
6. **Exercises** - Practice problems
