# FEAT-RL2: From-scratch NumPy plus Stable-Baselines3 dual implementation per lesson

## Kano
must-have

## Context
Each algorithm lesson pairs a from-scratch implementation (NumPy, or PyTorch for deep methods) with a production-framework reference (Stable-Baselines3), so learners see both the mechanics and the professional tool.

## Acceptance Criteria
- [ ] Each algorithm lesson ships a from-scratch implementation of its core algorithm
- [ ] Each algorithm lesson reproduces results with Stable-Baselines3 (or an equivalent framework)
- [ ] From-scratch and framework results are compared on the same environment
- [ ] API differences between the from-scratch and framework versions are documented
