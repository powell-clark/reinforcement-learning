## Context
Every notebook in the curriculum must open and run top-to-bottom on Google Colab's free tier, with visualizations rendering inline. This is the accessibility promise of the series.

## Acceptance Criteria
- [ ] Each notebook runs top-to-bottom on Colab free tier with no local-only dependencies
- [ ] A pip-install cell provisions Gymnasium, PyTorch, and Stable-Baselines3 as needed
- [ ] Plots and environment renders display inline in the notebook
- [ ] Long-running training cells expose a reduced-timestep quick-run path
