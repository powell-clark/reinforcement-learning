# STORY-RL1: Introduction to RL

## User Story
I want a first-principles introduction to reinforcement learning so that I understand the agent-environment loop before touching algorithms.

## Context
Lesson 0 — the RL paradigm. Frames RL against supervised/unsupervised learning and sets up the tooling every later lesson depends on.

## Acceptance Criteria
- [ ] Reader can explain states, actions, rewards, policies, and the exploration/exploitation tradeoff
- [ ] RL is contrasted clearly with supervised and unsupervised learning
- [ ] Model-based vs model-free and value-based vs policy-based taxonomies are introduced
- [ ] A working Gymnasium environment loop runs end to end in Colab
